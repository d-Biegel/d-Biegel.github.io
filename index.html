About Me

I'm a software developer with a unique passion bridging two worlds: legacy mainframe systems and modern game development. I earned my B.S. in Interactive Media & Game Design from SUNY Polytechnic Institute (Summa Cum Laude, 3.96 GPA), along with an A.S. in Computer Information Systems from Hudson Valley Community College.

My career interests span the full spectrum of software development—from exploring COBOL, CICS, and DB2 on z/OS systems to building games and applications with modern technologies like Unity and C#. This diverse background gives me a distinctive perspective on software architecture, system design, and problem-solving.

I'm excited about learning new technologies, tackling challenging problems, and creating impactful software solutions. Check out my projects and skills to see what I've been working on.

Featured Projects

Digital Board Game Adaptation

Technologies: Unity, C#

A collaborative academic project to adapt a physical board game into a digital experience. I programmed core gameplay systems including combat mechanics, raiding features, and comprehensive inventory management. Worked within an agile environment with weekly Scrum meetings to ensure effective team collaboration and iterative development.

Grocery Store System Redesign

Type: Systems Analysis & Design

Conducted a complete systems analysis and redesign for a grocery store operation. Deliverables included stakeholder registers, entity-relationship diagrams (ERDs), domain models, activity diagrams, use case diagrams, and comprehensive system vision documentation. This project strengthened my understanding of business requirements analysis and software design patterns.
